# Training Session

## What is this domain
Домен, отвечающий за проведение тренировочных сессий, в которых пользователь взаимодействует с AI-симуляцией (ролевой диалог, кейс, сценарий). Управляет жизненным циклом сессии от старта до завершения и фиксации результата.

## Why it exists
Это ядро продукта — без сессии нет тренировки. Домен инкапсулирует логику проведения, ограничения по времени/ходам и сбор результатов. Отделён от оценки и каталога сценариев, чтобы каждый домен мог развиваться независимо.

## Boundaries

### Inside this domain
- Создание и запуск сессии
- Обмен сообщениями (ходы пользователя и AI)
- Ограничения сессии (таймер, лимит ходов)
- Завершение сессии и фиксация outcome

### Outside this domain (explicitly)
- Каталог и редактирование сценариев (Scenario Catalog)
- Оценка качества ответов и обратная связь (Assessment)
- Профиль и прогресс ученика (Learner Progress)
- Биллинг и подписки (Billing)
- Генерация AI-ответов (LLM Gateway)

## Key use cases
- Ученик начинает тренировку по выбранному сценарию
- Ученик ведёт диалог с AI-фасилитатором
- Сессия автоматически завершается при достижении лимита ходов
- Сессия завершается по таймауту при неактивности
- Ученик досрочно завершает сессию вручную
